# 示波器信号发生器上位机

基于 WebSocket 的实时上位机界面，用于控制和显示 STM32 示波器信号发生器的数据。

## 功能特性

### 📊 示波器功能
- 实时波形显示（Chart.js）
- 自动/手动缩放
- 频率、电压测量
- 采样率和缓冲区配置

### 🔊 信号发生器功能
- 支持 5 种波形：正弦、方波、三角、锯齿、直流
- 频率和幅度可调
- 一键启停控制

### 💻 控制台功能
- 串口命令发送
- 快捷命令按钮
- 实时日志输出

## 安装步骤

### 1. 安装依赖

```bash
cd upper-computer
npm install
```

### 2. 启动服务器

```bash
npm start
```

### 3. 打开浏览器

访问 http://localhost:3000

## 使用说明

### 连接设备

1. 将 STM32 开发板通过 USB 转串口连接到电脑
2. 在上位机界面选择对应的串口
3. 选择波特率（默认 115200）
4. 点击"连接"按钮

### 使用示波器

1. 连接设备后，点击"开始采集"
2. 波形将实时显示在图表中
3. 可以调整采样率和缓冲区大小
4. 测量数据会自动更新

### 使用信号发生器

1. 选择波形类型
2. 设置频率和幅度
3. 点击"启动信号源"
4. 信号将从 DAC 输出

### 命令控制

可以在命令控制台输入以下命令：

- `help` - 显示帮助
- `status` - 显示设备状态
- `tasks` - 显示 FreeRTOS 任务
- `memory` - 显示内存使用
- `version` - 显示固件版本
- `errors` - 显示错误日志
- `scope start/stop` - 控制示波器
- `sig start/stop` - 控制信号发生器
- `reset` - 重启设备

## 数据格式

### 串口数据格式

波形数据：
```
WAVE:0A1B0C1D...（16进制采样值）
```

频率数据：
```
FREQ:1234.56
```

电压数据：
```
VOLT:0.123,3.210,3.087（最小,最大,峰峰值）
```

状态数据：
```
STATUS:{"sample_rate":1000000,"buffer_size":1024}
```

## 常见问题

### 串口无法连接

1. 检查串口是否被其他程序占用
2. 确认波特率设置正确
3. 检查 USB 转串口驱动是否安装

### 波形不显示

1. 确认设备已连接并开始采集
2. 检查浏览器控制台是否有错误
3. 确认 WebSocket 连接正常

### 数据不同步

1. 点击"清除"按钮清除旧数据
2. 重新开始采集
3. 检查采样率设置
