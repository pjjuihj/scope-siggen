# 按键操作指南

## 硬件配置

| 引脚 | 功能 | 按下电平 |
|------|------|----------|
| PA0 | 页面切换键 | 高电平 |
| PA1 | 菜单选择键 | 高电平 |

## 按键功能

### 菜单页面 (PAGE_MENU)

| 按键 | 功能 |
|------|------|
| PA0 短按 | 切换到下一个页面 |
| PA0 长按 | 确认选择当前菜单项 |
| PA1 短按 | 移动到下一个菜单选项 |

### 其他页面

| 按键 | 功能 |
|------|------|
| PA0 短按 | 切换到下一个页面 |
| PA0 长按 | 保存配置到 Flash |
| PA1 短按 | 无功能 |

## 页面循环顺序

```
OSCOPE → SIGGEN → SYSINFO → MENU → MESSAGE → OTA → OSCOPE
```

## 菜单选项

| 选项 | 功能 |
|------|------|
| Oscilloscope | 切换到示波器页面 |
| Signal Gen | 切换到信号发生器页面 |
| System Info | 切换到系统信息页面 |
| About | 显示版本信息 |

## 使用示例

1. **切换页面**：短按 PA0
2. **进入菜单**：短按 PA0 直到显示菜单页面
3. **选择菜单项**：短按 PA1 移动选择
4. **确认选择**：长按 PA0
5. **保存配置**：在非菜单页面长按 PA0

## 操作记录

在 MESSAGE 页面可以查看最近 5 条操作记录，包括：
- 页面切换
- 菜单操作
- 配置保存
