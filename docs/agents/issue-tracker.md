# Issue Tracker

## Backend

GitHub Issues

- Remote: https://github.com/pjjuihj/scope-siggen.git
- CLI: `gh`
- PRs as request surface: yes

## Workflow

Skills (`to-issues`, `triage`, `to-prd`, `qa`) read from and write to GitHub Issues using the `gh` CLI.

External PRs are pulled into the triage queue and processed with the same labels and states as issues. Collaborators' in-flight PRs are left alone.
