## 技术收获

### 1. STM32 Boot0 引脚
- Boot0 引脚启动后可作为 GPIO (PB0) 使用
- 板子上通常有硬件下拉电阻，接 GND 无法检测按下
- 需要根据实际硬件接线选择上拉或下拉配置

### 2. 按键检测
- 下拉电阻配置：按下为高电平
- 上拉电阻配置：按下为低电平
- 需要先用万用表测量确认

### 3. Hugo 静态网站
- baseURL 配置影响所有链接
- 相对路径 `'/'` 支持多平台部署
- GitHub Pages 需要正确的 baseURL

### 4. Git 操作
- `git pull --no-rebase` 避免复杂的合并
- 推送前先拉取远程更改
- 使用 `.gitignore` 排除构建产物
